Local Music to Spotify Matcher

Github Repo

I still use local media to listen to music, but sometimes I want to share my library with others. This program searches for flac, mp3, and m4a files in a given directory, then adds matching Spotify songs to the user's Spotify library. It's written in Python 3 and uses Spotipy Spotify API and Mutagen to grab file info. The code for this project can be found on my Github. The next step is to use this program's functionality to implement a m3u playlist to Spotify playlist program.

March 2020

Am I at Risk?

Devpost Link

Am I at Risk? aims to inform users if they are at risk of Covid-19 based on their travel location history. With an easy to use interface, users can list the places they've been along with the date and time of day. Am I at Risk? will check if any of the locations have been visited by a Covid-19 carrier during or shortly before the user visited the area.
This project was made in a team for Winhacks 2020 and won Domain.com's "Best Domain Registered with Domain.com" award. The frontend was built using HTML, CSS, JavaScript, and Bootstrap. We also made use of Google's Geocoding API to allow for easier location input. For the backend, we used PHP and MySQL.

Sept. - Nov. 2018

AWS MySQL Database and API

Github Repo

In my Intro to Software Engineering class, I was tasked with creating a database for my team's android app. This database held image files which were to be served to the android app. To do this, I used a MySQL database hosted on AWS RDS. For database access, I created a Java Maven project and stored it on AWS Lambda. This Lambda function was then triggered using AWS API Gateway. To connect the database to the Android app, I created an iterator class in Java which retrieved image URLs from the database. This iterator class was then implemented into the app.
