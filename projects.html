Local music to Spotify matcher

local-to-spotify

I still use local media to listen to music, but sometimes I want to share my library with others. This program searches for flac, mp3, and m4a files in a given directory, then adds matching Spotify songs to the user's Spotify library. The next step is to use this program's functionality to implement a m3u playlist to Spotify playlist program.
